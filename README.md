# D&D Speedster Damage Calculator

A simple web calculator for homebrew D&D speedster class damage based on movement speed.

## How to Use

1. Open `index.html` in VS Code
2. Right-click and select "Open with Live Server"
3. Enter your character's movement speed and level
4. Click "Calculate Damage" to see results

## Current Formula

Damage = (Speed รท 10) + (Level รท 2)

## Customization

Edit the `calculateDamage()` function in `script.js` to modify the damage formula.
